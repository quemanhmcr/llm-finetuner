# Contributing to LLM Reasoning Finetuner

Thank you for your interest in contributing to this project! We welcome contributions from the community to help improve this research framework.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button at the top right of the repository page.
2.  **Clone your fork**: `git clone https://github.com/your-username/llm-finetuner.git`
3.  **Create a branch**: `git checkout -b feature/your-feature-name`
4.  **Make your changes**: Implement your feature or fix.
5.  **Commit your changes**: `git commit -m "feat: add new feature"` (Please follow [Conventional Commits](https://www.conventionalcommits.org/))
6.  **Push to your fork**: `git push origin feature/your-feature-name`
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".

## Code Style

*   We use `black` for code formatting.
*   We use `isort` for import sorting.
*   Please ensure your code passes all tests before submitting.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue.
